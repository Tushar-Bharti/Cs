# Write a C program to find the average of two numbers.

## Solution:

>```c
>#include <stdio.h>
>
>int main(void)
>{
>    float a, b;
>    printf("Enter number 1 -> ");
>    scanf("%f", &a);
>
>    printf("Enter number 2 -> ");
>    scanf("%f", &b);
>
>    printf("Average of %f (number 1) & %f (number 2) is -> %f", a, b, (a + b) / 2);
>    return 0;
>}
>```
