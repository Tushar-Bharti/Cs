# Write a C program to calculate the area of a rectangle.

## Solution:
>```c
>#include <stdio.h>
>
>int main(void) {
>    int l,b;
>    printf("Enter the length of rectangle ->");
>    scanf("%d",&l);
>
>    printf("Enter the breath of rectangle ->");
>    scanf("%d", &b);
>
>    printf("Area of rectangle is -> %d",l*b);
>    return 0;
>}
>```
